## Pre-Submission Tasks

1. **Deploy Contracts to BNB Testnet**
   - Run `npm run deploy:testnet`
   - Verify contracts on BSCScan
   - Save contract addresses

2. **Update Frontend with Contract Addresses**
   - Update `frontend/lib/contracts.ts` with deployed addresses
   - Update `.env.local` with contract addresses

3. **Deploy Frontend**
   - Deploy to Vercel or Netlify
   - Set environment variables
   - Test all functionality

4. **Record Demo Video**
   - Write script (5 minutes max)
   - Record screen (HD quality)
   - Record audio (clear narration)
   - Edit video (subtitles, transitions)
   - Upload to YouTube (public or unlisted)

5. **Fill Team Info**
   - Write 150-word team description
   - Add to README.md

6. **Final Testing**
   - Test all features on live deployment
   - Verify all links work
   - Check for bugs

7. **Submit to DoraHacks**
   - Complete "Submit Buidl" form
   - Add all required links
   - Submit before deadline
